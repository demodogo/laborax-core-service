# Vista de Componentes

## `platform-core-service`

- API de Auth
- Gestión de Usuarios
- Gestión de Memberships y RBAC
- Gestión de Tenants y Companies
- Gestión de Contratos
- Auth de Servicios Internos
- Outbox Publisher

## `workforce-registry-service`

- Gestión de Workers
- Gestión de Vehicles
- Gestión de Assignments
- API de Consulta del Registro
- Outbox Publisher

## `accreditation-service`

- Gestión de Requisitos
- API de Envío
- Workflow de Revisión
- Motor de Acreditación
- Alert Scheduler
- Reconciler

## `access-control-service`

- Gestión de Locations
- Gestión de Devices
- Gestión de Credenciales
- Motor de Decisión de Acceso
- Journal de Access Events
- Event Consumer
- Reconciler
