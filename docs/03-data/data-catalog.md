# Catálogo de Datos

| Entidad | Descripción | Servicio Owner |
| --- | --- | --- |
| `tenant` | Alcance de cliente o cliente top-level | `platform-core-service` |
| `company` | Nodo legal u operacional de empresa | `platform-core-service` |
| `worker` | Registro canónico de persona laboral | `workforce-registry-service` |
| `vehicle` | Registro canónico de vehículo | `workforce-registry-service` |
| `workerCompanyAssignment` | Vínculo entre worker y company | `workforce-registry-service` |
| `requirementDefinition` | Definición de regla de acreditación | `accreditation-service` |
| `documentSubmission` | Evidencia enviada para acreditación | `accreditation-service` |
| `accreditationStatus` | Estado calculado de acreditación del sujeto | `accreditation-service` |
| `accessCredential` | Credencial física o lógica usada en acceso | `access-control-service` |
| `accessEvent` | Evento registrado durante decisión de acceso | `access-control-service` |
| `compliancePeriod` | Ciclo mensual o periódico de cumplimiento | `contractor-compliance-service` |
| `issuedCertificate` | Certificado laboral emitido | `certification-service` |
| `employeeProfile` | Perfil laboral extendido | `people-hr-service` |
| `karinCase` | Caso confidencial Ley Karin | `karin-case-service` |
