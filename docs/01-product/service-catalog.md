# Catálogo de Servicios

## Servicios de Negocio

| Servicio | Responsabilidad Principal | Cobertura Principal |
| --- | --- | --- |
| `platform-core-service` | Identidad, auth, RBAC, tenants, companies, contratos | Todos |
| `workforce-registry-service` | Registro maestro de trabajadores y vehículos | Todos los productos relacionados con workforce |
| `access-control-service` | Decisiones de acceso, sesiones, dispositivos, checkpoints | SCA |
| `accreditation-service` | Requisitos, revisión documental, estado de acreditación | Acreditax |
| `contractor-compliance-service` | Períodos de cumplimiento, observaciones, reportería ejecutiva | SCC |
| `certification-service` | Workflow de certificación y emisión | Certificax |
| `people-hr-service` | Beneficios, capacitación, estructura, perfil laboral | Gestionax |
| `karin-case-service` | Denuncias, investigaciones, plazos y cierre | SmartKarin |

## Servicios Compartidos

| Servicio | Responsabilidad |
| --- | --- |
| `portal-bff` | Agregación para web y composición con contexto de sesión |
| `notification-service` | Correos, recordatorios y notificaciones |
| `document-storage-service` | Almacenamiento seguro y entrega controlada de documentos |
