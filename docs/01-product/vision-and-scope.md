# Visión y Alcance

## Visión

Laborax es una plataforma empresarial multiproducto para cumplimiento de contratistas, acreditación de trabajadores y vehículos, control de acceso físico, certificación laboral, gestión de recursos humanos y gestión de casos legales.

## Objetivos del Producto

- Proveer una experiencia de plataforma única para múltiples productos.
- Centralizar identidad y autorización transversal.
- Eliminar duplicidad de datos maestros de trabajadores, vehículos y entidades organizacionales.
- Permitir evolución independiente de cada capacidad de negocio.
- Soportar auditoría, trazabilidad regulatoria e integraciones controladas.

## Dentro del Alcance

- Identidad de plataforma y modelo de tenants
- Cumplimiento de contratistas
- Registro maestro de trabajadores y vehículos
- Flujos de acreditación
- Control de acceso
- Certificación laboral
- Gestión de recursos humanos
- Gestión de casos Ley Karin
- Capacidades compartidas de notificación, almacenamiento documental y auditoría

## Fuera del Alcance

- Procesamiento de remuneraciones
- Contabilidad ERP general
- Implementación de data warehouse externo
- Desarrollo de firmware para hardware de terceros
- Federación IAM externa fuera de los contratos definidos

## Productos de Negocio

| Producto | Propósito |
| --- | --- |
| `Laborax SCC` | Gestión mensual del cumplimiento laboral y regulatorio de contratistas |
| `Laborax SCA` | Control de acceso físico en tiempo real |
| `Acreditax` | Acreditación documental de trabajadores y vehículos |
| `Certificax` | Flujo de certificación laboral y emisión de certificados |
| `Gestionax` | Registros de RRHH, capacitación, estructura y beneficios |
| `SmartKarin` | Gestión de denuncias y casos bajo Ley Karin |
