# Observabilidad

## Señales Requeridas

- Logs estructurados
- Métricas
- Trazas distribuidas
- Resultados de ejecución de jobs
- Lag de colas y resultados de reconciliación

## Dashboards Mínimos

- Latencia API y tasa de error
- Éxito o falla de autenticación
- Lag de publicación de eventos
- Lag de consumo de eventos
- Éxito o falla de reconciliación
- Carga de vencimientos de acreditación
- Throughput de decisiones de acceso
